## 📋 Sobre o projeto

Trabalho prático da disciplina de **Algoritmos II**: um sistema de gestão hoteleira escrito em **linguagem C**, com cadastros (CRUDs) de hóspedes, quartos, reservas, etc., e geração de relatórios.

## 🛠️ Tecnologias

- **C** (ANSI C)

## 🎯 Funcionalidades

- Cadastro, listagem, edição e remoção de **hóspedes**
- Gestão de **quartos** e **reservas**
- Geração de **relatórios** (ocupação, faturamento, etc.)
- Persistência em arquivos `.txt` ou `.dat`

## 🚀 Como compilar e executar

```bash
gcc main.c -o hotel
./hotel
```

> Caso o repositório esteja com a estrutura modular, compile todos os `.c`:
> ```bash
> gcc *.c -o hotel
> ```

## 🎯 Conceitos aplicados

- Estruturas (`struct`)
- Manipulação de arquivos
- Alocação dinâmica de memória
- Modularização em C
